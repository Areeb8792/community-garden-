# 🌿 Community Garden Website

## 📌 Overview
The Community Garden Website is a web-based platform designed to connect local residents, volunteers, and organizers. It helps manage gardening activities, events, and shared garden spaces while promoting sustainability and community engagement.

---

## 🚀 Features
- 👤 User Registration & Login  
- 🌱 Garden Plot Information  
- 📅 Event & Workshop Management  
- 🤝 Volunteer Participation  
- 📢 Announcements & Updates  
- 📬 Contact & Feedback System  

---

## 🛠️ Technologies Used
- Frontend: HTML, CSS, Tailwind CSS  
- Backend: PHP / Node.js *(update as per your project)*  
- Database: MySQL
